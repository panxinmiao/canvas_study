# canvas_tetris
Simple tetris game by pure javascript


最早是我在某个项目的web页面里埋下的彩蛋，按住Ctrl+Alt，并依次按下↑↑↓↓←←→→，会触发该小游戏

直接触发后的效果 : http://panxinmiao.github.io/canvas_tetris/

控制按键:

←   :   move left

→   :   move right

↓   :   move down

↑   :   change

space : drop directly

,   :   speed down

.   :   speed up

s   :   show/hide shadow

g   :   show/hide grid
